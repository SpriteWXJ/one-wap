# 山寨ONE app


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

### 技术栈
```
基于vue2 + vue-router + vuex + vue-axios重写vue版山寨ONE，使用webpack打包
```
